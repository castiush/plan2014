Plan de trabajo
2014
Adaptar Huayra Linux
A las necesidades de la provincia...
De cada nivel.
(Inicial, primaria...)
Y cada modalidad.
(Especial, adultos...)
¿Cómo?
  • Metapaquetes.
  • Documentación.
  • Postinstall.
  • Software propio.
Metapaquetes:
Permiten, en un sólo paso, instalar una serie de programas.
Documentación:
Adaptando la aplicación "Huayra primeros pasos" en cada caso.
Postinstall:
Permite en un sólo paso aplicar una serie de configuraciones.
Software propio:
Adaptar software local para hacerlo llegar a todo el país como parte de Huayra Linux.
¿El proceso?
Relevamiento exhaustivo.
Desarrollo colaborativo y transparente.
Evolución y mantenimiento en comunidad.
Colaboración +
transparencia...
Generan un círculo virtuoso.
¿Un ejemplo?
Educación especial.
Cada tipo de discapacidad implica requerimientos distintos.
Lectores de pantalla, lupas, alto contraste, impresoras braile, estimulación, simplificación del escritorio...
Cada una requiere horas de trabajo con cada netbook para volverla "usable".
Mediante las adaptaciones propuestas,
se reducen tiempo y esfuerzo y se comparte lo aprendido.
Todos colaboramos en la mejora de una solución sustentable que evoluciona con el tiempo.
¿Con qué contamos?
Referentes técnicos con años de experiencia en su nivel / modalidad.
El trabajo realizado por el equipo que desarrolló Educadito Linux  pensado para especial y primaria.
La experiencia de equipos que desarrollan distribuciones Linux específicas.
Experiencia en el desarrollo de software libre en la provincia.
Una comunidad de más de 3.000.000 de usuarios de las netbook de P.C.I.
Un equipo nacional que brinda soporte técnico y promueve desarrollos de este tipo.
Una dirección de informática con cultura de software libre.
Resumiendo:
Un esfuerzo colaborativo,
de Tierra del Fuego con el país
para producir
soluciones sustentables y soberanas.
